Para uso diário com interface gráfica moderna e aplicativos essenciais pré‑instalados.

Instalação

Crie um pendrive bootável e siga o instalador.

Linux (dd)

# Substitua /dev/sdX pelo seu dispositivo
echo "ATENÇÃO: isso apaga o pendrive." 
sudo dd if=FlyLinux-Workstation-x86_64.iso of=/dev/sdX bs=4M status=progress conv=fsync

Windows (Rufus)

  1. Baixe a ISO do Fly Linux
  2. Abra o Rufus e selecione seu pendrive
  3. Escolha a ISO e clique em Start

Alternativa (Ventoy)

  1. Instale o Ventoy no pendrive
  2. Copie a ISO do Fly Linux para o pendrive
  3. Inicie o PC pelo pendrive e selecione a ISO

Perguntas frequentes

O Fly Linux é gratuito?

Sim. O sistema é livre e gratuito. Você pode baixar, usar e redistribuir seguindo as licenças aplicáveis.

Quais arquiteturas são suportadas?

Atualmente, x86_64 (64‑bit). Outras arquiteturas podem ser consideradas conforme a demanda.

Como colaborar com o projeto?

Participe no fórum, relate problemas, envie sugestões e contribua com documentação e código.
